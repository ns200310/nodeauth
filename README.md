# NodeAuth
 NodeJS + MongoDB + ExpressJs, User Authentication and Management System
 
## Frameworks and Languages
- Serverside: NodeJS
- Database: MongoDB
- App Framework: ExpressJS
- View Engine: PUG


## Dependencies
- bcryptjs 2.3.0
- body-parser 1.19.0
- connect-flash
- cookie-parser 1.3.5
- debug 4.3.2
- express 4.17.1
- express-messages
- express-session 1.13.0
- express-validator 3.0.0
- mongodb
- mongoose
- morgan 1.10.0
- multer
- passport
- passport-http
- passport-local
- pug 3.0.2
- serve-favicon 2.5.0

## Installation
### Install MongoDB
### Install NodeJS
### Install NodeAuth
```
git clone https://github.com/sathninduk/nodeauth.git nodeauth
```
```
cd nodeauth
```
```
npm install
```
```
npm start
```
http://localhost:3000
